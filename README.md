# 1. AI Chatbot (OpenAI / LangChain Basics)

## Overview
Built a simple AI chatbot using the OpenAI API that can hold natural conversations with users. The project focuses on understanding how large language models process prompts and generate contextual responses.

## Key Features
- Conversational AI using OpenAI API
- Basic prompt engineering
- Simple chat loop interface
- Context-aware responses within a session

## Learning Outcomes
- How LLMs process input/output
- Basics of API integration
- Introduction to prompt design



# 2. AI Tool-Using Agent

## Overview
Built an AI agent capable of using external tools to complete tasks beyond basic conversation. The agent decides when to use tools like calculators to solve problems more accurately.

## Key Features
- Tool integration (e.g., calculator functions)
- Multi-step reasoning with LLMs
- Structured agent workflow
- Dynamic decision-making (tool vs. response)

## Learning Outcomes
- Tool-augmented LLM behavior
- Basic agent architecture (think → act → respond)
- Improving LLM reliability with external functions

# 3. AI Memory Chatbot

## Overview
Built an AI chatbot with memory capabilities, allowing it to remember previous user inputs and maintain context across multiple conversations.

## Key Features
- Short-term conversation memory
- Context retention across messages
- Improved response relevance over time
- Persistent chat structure using stored history

## Learning Outcomes
- How memory is implemented in LLM systems
- Managing conversation state
- Improving coherence in multi-turn dialogue systems
